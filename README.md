Aurora International School – Interactive Website
Aurora International School is a modern, cinematic and interactive school website project created using HTML, CSS and JavaScript. The goal of this project is to present a premium school website experience with clean design, smooth animations and an image-focused campus exploration section. The website follows a dark, professional theme with subtle motion effects to make it visually appealing while still remaining simple and easy to understand for academic and college project use.
This project includes a custom user interface with smooth hover effects, section reveal animations on scroll and a square-style custom cursor that reacts to user interaction. The homepage introduces the school, its values, academic structure, houses and facilities, while the Explore Campus page focuses mainly on visuals with short, clear descriptions instead of long paragraphs. This makes the website feel modern, engaging and professional.
Key features of the project include a cinematic dark theme, responsive layout, interactive navigation bar, custom animated cursor, smooth hover transitions, scroll-based reveal animations and a separate Explore Campus page with image-based facility cards such as Library, Science Lab, Computer Lab, Sports Complex and Auditorium. The Admissions Open button is intentionally designed to show an informational pop-up message stating that online registrations are currently closed and advising users to contact the school office directly. This behaviour keeps the project realistic and suitable for demonstration purposes.
The project structure is simple and well organised. The root folder contains index.html as the main homepage and campus.html as the Explore Campus page. All images, videos and audio files are stored inside the assets folder. Styling files are placed in the css folder, including style.css, intro.css and cursor.css. JavaScript functionality such as animations, cursor behaviour, intro handling and background effects are managed inside the js folder. This structure makes the project easy to understand, modify and explain during viva or presentations.
To run the project, simply open the folder in Visual Studio Code and use Live Server to launch index.html for the best experience. The project can also be opened directly in a browser by double-clicking the HTML file, although some animations and media interactions may work better when served locally. Navigation between pages works through standard anchor links, and no backend or database is required as this is a front-end-only project.
While using or modifying this project, it is recommended not to rename files or folders without updating their paths inside the HTML files. Images should remain in their respective directories to avoid broken layouts. Backend features such as forms, databases or login systems are intentionally not included, as the purpose of the project is to demonstrate front-end design, layout and interactivity rather than full-stack functionality.
Overall, the Aurora International School website project is designed to look premium, modern and professional while remaining simple enough for academic use. It demonstrates strong front-end fundamentals, clean design thinking and interactive UI concepts, making it suitable for college submissions, portfolio showcase and front-end practice.
